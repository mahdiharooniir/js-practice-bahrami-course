• زبان نشانه‌گذاری ابرمتن (به انگلیسی: Hyper Text Markup Language ) زبان نشانه‌گذاری استاندارد برای اسنادی‌ست که برای نمایش‌دادن در وب طراحی شده اند. اچ‌تی‌ام‌ال زبانی‌ست که محتوا و ساختار صفحات وب را تعریف و توصیف می‌کند.

  • CSS زبان طراحی صفحات وب است که برای استایل‌دهی و زیبایی بخش دادن به HTML استفاده می‌شود.

  • JavaScript زبان برنامه‌نویسی سمت کلاینت است که برای پویا کردن صفحات وب و تعامل با کاربر استفاده می‌شود.

  • DOM مخفف Document Object Model است و یک رابط برنامه‌نویسی برای دسترسی و تغییر محتوای صفحات وب است.
